2. Possessives (Posesivos)

Usamos estas palabras para decir a quién le pertenece algo:

This is my... Esto es mi... (ej: This is my book / Este es mi libro)
This is your... Esto es tu... (ej: This is your pen / Esta es tu pluma)

❓ ¿Cómo puedo decir "Esto es"?

Para señalar o presentar algo, usamos siempre:

THIS IS...

Pronunciación aproximada: "Dis is"
